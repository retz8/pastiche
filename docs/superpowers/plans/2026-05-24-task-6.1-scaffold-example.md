- [ ] **Step 2: Create client-side providers component**

Primer's `ThemeProvider` and `BaseStyles` use React context and must run on the client. Create a `"use client"` wrapper.

```tsx
// src/app/providers.tsx
"use client";

import { ThemeProvider, BaseStyles } from "@primer/react";

export function Providers({ children }: { children: React.ReactNode }) {
  return (
    <ThemeProvider colorMode="day">
      <BaseStyles>{children}</BaseStyles>
    </ThemeProvider>
  );
}
```

- [ ] **Step 3: Update `layout.tsx` to use Providers**

- `../..` → `examples/github-primer-react/.claude/`
